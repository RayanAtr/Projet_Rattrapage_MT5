# Projet_Rattrapage_MT5
# FlexBook — Installation

## Prérequis

- Node.js >= 18  
- npm ou yarn

## Installation

```bash
cd react-project
npm install
```

## Lancer en local
``` bash 
npm run dev
```
## Build
```bash
npm run build
npm run preview
```
## Stack technique
- React + Typescript + Vite
- Supabase
- Recharts

## Outils externe (API)
- Les images proviennent de l'api Lorem Picsum
- Les QR codes sont générés via https://goqr.me/api/
